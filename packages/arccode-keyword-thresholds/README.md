# language-keywords

A collection of keywords for various programming languages.

## Installation

```bash
npm install language-keywords
```

## Usage

```typescript
import { ruby } from 'language-keywords';

console.log(ruby); // ['alias', 'and', 'begin', ...]
```

## Supported Languages

- C
- C++
- C#
- Go
- Java
- JavaScript
- PHP
- Python
- Ruby
- TypeScript
